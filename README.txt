1. run "composer update"
2. .env указать свои параметры подключения к бд
3. config/app.php указать необходимую timezone
4. "php artisan migrate"
5. "php artisan serve"
6. Чтобы проверять разных юзеров, достаточно в url указывать token юзера
7. Ссылка копируется в буфер обмена
8. При деактивации ссылки в базе для данного юзера просто обновляется время для активности данной ссылки
9. ajax реализован для кнопок "I'm Feeling Lucky" и для "History"
