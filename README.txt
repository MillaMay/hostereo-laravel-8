1. run "composer update"
2. .env указать свои параметры подключения к бд
3. config/app.php указать необходимую timezone
4. "php artisan migrate"
5. Чтобы проверять разных юзеров, достаточно в url указывать token юзера
6. Ссылка копируется в буфер обмена
7. При деактивации ссылки в базе для данного юзера просто обновляется время для активности данной ссылки
8. ajax реализован для кнопок "I'm Feeling Lucky" и для "History"
